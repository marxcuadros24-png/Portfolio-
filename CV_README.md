# Marx Cuadros - CV Placeholder

Este archivo es un placeholder para tu CV.

Para agregar tu CV real:
1. Guarda tu CV en formato PDF
2. Colócalo en la carpeta `public` con el nombre `Marx_Cuadros_CV.pdf`
3. El botón de descarga ya está configurado para usar ese archivo

Mientras tanto, el botón aparecerá cuando agregues el archivo CV.
